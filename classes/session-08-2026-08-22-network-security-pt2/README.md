## About This Session

Part 2 goes beyond basic firewall rules and looks at more advanced network protection tools. You will study Azure DDoS Protection (defending against volumetric attacks that try to overwhelm your services), Azure Bastion (a secure way to connect to VMs without exposing RDP/SSH ports to the internet), Private Endpoints (keeping traffic to Azure services completely off the public internet), and Azure Network Watcher (diagnosing network issues and capturing traffic).

**By the end of this session you will be able to:**
- Explain what DDoS attacks are and how Azure DDoS Protection helps
- Use Azure Bastion to securely access virtual machines
- Configure Private Endpoints to eliminate public exposure of PaaS services
- Use Network Watcher to diagnose connectivity problems
