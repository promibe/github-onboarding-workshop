## About This Session

Network security in Azure is about controlling what traffic can flow in and out of your cloud resources. Just like a physical building has doors, locks, and security guards, your cloud network needs rules about what is allowed through.

Part 1 covers the core building blocks: Virtual Networks (VNets), Network Security Groups (NSGs — firewall rules on network interfaces and subnets), Azure Firewall (a managed, centralised firewall), and how to segment your network so that a breach in one part cannot spread to everything else.

**By the end of this session you will be able to:**
- Explain how Azure Virtual Networks and subnets work
- Create and apply Network Security Group rules
- Describe the difference between NSGs and Azure Firewall
- Apply network segmentation principles to limit blast radius
