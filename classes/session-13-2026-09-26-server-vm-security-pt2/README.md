## About This Session

Part 2 focuses on keeping VMs patched and their disks encrypted. An unpatched VM is one of the simplest ways for an attacker to get in. Azure Update Manager automates patch deployment, while Azure Disk Encryption and Trusted Launch protect the data on the VM and its boot process.

You will also look at Microsoft Defender Vulnerability Management — a tool that continuously scans your VMs for known vulnerabilities and prioritises which ones to fix first.

**By the end of this session you will be able to:**
- Use Azure Update Manager to automate VM patching
- Enable Azure Disk Encryption to protect data at rest on VMs
- Explain how Trusted Launch protects against boot-level attacks
- Interpret vulnerability assessment results from Defender for Servers
