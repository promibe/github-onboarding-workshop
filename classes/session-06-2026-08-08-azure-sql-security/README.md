## About This Session

Azure SQL Database is Microsoft's fully managed cloud database. Protecting the data inside it — especially personal or financial data — is a key SC-500 skill.

This session covers authentication options (Entra-only vs SQL logins), firewall rules, Advanced Data Security (vulnerability assessments and threat detection), Always Encrypted (keeping data encrypted even from database admins), and auditing so you have a log of every query run against sensitive tables.

**By the end of this session you will be able to:**
- Configure authentication and firewall rules for Azure SQL
- Enable Microsoft Defender for SQL to detect SQL injection and anomalous queries
- Describe how Always Encrypted protects sensitive columns
- Set up SQL auditing and understand the logs it produces
