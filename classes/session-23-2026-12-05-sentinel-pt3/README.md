## About This Session

The final Sentinel session covers the more advanced and powerful capabilities: Playbooks (automated response workflows built on Azure Logic Apps that can isolate a compromised machine, send a Teams alert, or create a ticket — automatically), Threat Hunting (proactively searching through logs for threats that have not triggered an alert), and Workbooks (dashboards that visualise your security data).

You will also look at UEBA (User and Entity Behaviour Analytics) — Sentinel's ability to detect when a user or device is acting unusually compared to their normal pattern.

**By the end of this session you will be able to:**
- Build a Sentinel Playbook to automate an incident response action
- Write basic KQL queries for threat hunting
- Create a Sentinel Workbook to visualise security trends
- Explain how UEBA detects insider threats and compromised accounts
