## About This Session

Part 2 moves from theory into configuration. You will look at how to secure the data that AI models train on and retrieve, how to lock down Azure OpenAI deployments (network access, content filters, managed identity authentication), and how to monitor AI systems for anomalous behaviour using Microsoft Defender for AI and Azure Monitor.

**By the end of this session you will be able to:**
- Configure network and identity controls for Azure OpenAI
- Apply content filtering policies to AI models
- Secure the data pipelines that feed AI applications
- Set up monitoring and alerting for AI workload security events
