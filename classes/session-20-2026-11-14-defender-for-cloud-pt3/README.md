## About This Session

The final Defender for Cloud session looks at extending protection beyond Azure. Many organisations use AWS and Google Cloud alongside Azure, and Defender for Cloud can monitor all of them from a single pane of glass through its multi-cloud connectors.

You will also look at DevOps security integration (connecting GitHub and Azure DevOps to Defender for Cloud), advanced CSPM features like cloud security graph queries, and how to build custom security standards and recommendations for your organisation.

**By the end of this session you will be able to:**
- Connect AWS and GCP environments to Defender for Cloud
- Use cloud security graph to query relationships between cloud resources
- Build custom compliance standards in Defender for Cloud
- Integrate DevOps repositories for supply chain visibility
