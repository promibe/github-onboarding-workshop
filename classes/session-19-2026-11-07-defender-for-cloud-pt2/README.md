## About This Session

Part 2 moves from posture assessment into active threat detection. Defender for Cloud's workload protection plans (Defender for Servers, Defender for Databases, Defender for Containers, etc.) monitor your running resources and generate security alerts when something suspicious happens.

You will learn how to enable these plans, understand the types of alerts they generate, investigate a security alert from first detection through to resolution, and use the Attack Path Analysis feature to see how an attacker might chain vulnerabilities together to reach a critical resource.

**By the end of this session you will be able to:**
- Enable Defender for Cloud workload protection plans
- Interpret and investigate Defender for Cloud security alerts
- Use Attack Path Analysis to find and close exploit chains
- Describe how Defender for Cloud integrates with Microsoft Sentinel
