## About This Session

Microsoft Defender for Cloud is your central security dashboard for everything running in Azure (and beyond). It continuously assesses your cloud environment, gives you a Secure Score, and tells you exactly what to fix to improve your security posture.

Part 1 introduces Cloud Security Posture Management (CSPM): what it is, how Defender for Cloud calculates your Secure Score, how to navigate the Recommendations blade, and how to use the regulatory compliance dashboard to see where you stand against frameworks like NIST, PCI-DSS, and ISO 27001.

**By the end of this session you will be able to:**
- Explain what CSPM means and why Defender for Cloud is central to SC-500
- Navigate the Secure Score and understand how recommendations are weighted
- Use the Regulatory Compliance dashboard to assess compliance posture
- Prioritise and remediate Defender for Cloud recommendations
